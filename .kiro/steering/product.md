# Product Overview

Ghost Stories is a WordPress plugin that brings Instagram-style stories to WordPress websites. It enables content creators to publish ephemeral, engaging visual content with an immersive modal viewing experience.

## Core Features

- **Story Management**: Create stories with multiple images/videos through WordPress admin
- **Expiration System**: Stories automatically expire after a configurable time period (default 24 hours)
- **Gutenberg Block**: Insert story circles anywhere using the native block editor
- **Modal Viewer**: Full-screen, mobile-first story viewer with touch gestures
- **REST API**: Headless CMS support via `/wp-json/wp-stories/v1/stories` endpoints
- **Media Support**: Images (JPEG, PNG, GIF) and videos (MP4, WebM)

## User Experience

- **Frontend**: Story circles with gradient borders, tap to open modal viewer
- **Navigation**: Swipe/arrow keys for media, vertical swipe for story switching
- **Video Controls**: Play/pause, volume toggle, synchronized progress bars
- **Responsive**: Optimized for mobile, tablet, and desktop
- **Accessibility**: ARIA labels, keyboard navigation, screen reader support

## Target Audience

News sites, blogs, e-commerce stores, event websites, and any WordPress site wanting to increase engagement with ephemeral visual content.
